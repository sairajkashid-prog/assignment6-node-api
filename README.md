# 🚀 Assignment 6 - Node.js REST API

## 📌 Overview

This project is a simple **REST API built using Node.js and Express**.
It performs CRUD operations on a **Books collection**.

---

## ⚙️ Tech Stack

* Node.js
* Express.js
* Postman (for testing)

---

## 📂 Project Structure

```
rest-api/
│── node_modules/
│── server.js
│── package.json
│── package-lock.json
│── .gitignore
```

---

## ▶️ How to Run

```bash
npm install
node server.js
```

👉 Server runs on:

```
http://localhost:3000
```

---

## 📡 API Endpoints

### 🔹 GET all books

```
GET /api/books
```

---

### 🔹 GET single book

```
GET /api/books/:id
```

---

### 🔹 POST (Add book)

```
POST /api/books
```

**Body (JSON):**

```json
{
  "name": "Book Name",
  "author": "Author Name"
}
```

---

### 🔹 PATCH (Update book)

```
PATCH /api/books/:id
```

---

### 🔹 DELETE book

```
DELETE /api/books/:id
```

---
